# ER.gg

게임 플레이 데이터를 수집·집계하여 **통계 대시보드로 시각화**하는 Node + Vue 기반 프로젝트입니다.  
개인 프로젝트/학습용으로 제작되었으며, 프론트엔드(Vue)와 백엔드(Node)가 분리된 구조를 가정합니다.

> op.gg를 보방한 이터널리턴 전용 ER.gg

## Features

- 게임 데이터 통계 집계 (플레이 횟수, 점수, 추이 등)
- Vue 기반 대시보드 UI
- 차트/그래프를 통한 시각화
- Node 서버를 통한 데이터 제공 API

---

## Tech Stack

- **Frontend**: Vue.js
- **Backend**: Node.js
- **Package Manager**: npm
- **Visualization**: Chart/Graph Library (예: Chart.js 등)
  
## Project setup
```
npm install
```

### 서버 실행 (BackEnd, FrontEnd 실행해야 됨)
```
npm run serve
node api/server.js
```

### 접속

- `http://localhost:3000`
- 또는 Vite 기본: `http://localhost:5173`
